# IDEAS.md

Aggressive ideas for pivots, refactoring, re-architecting:
- **Real-Time Websockets:** Introduce a real-time chat overlay for specific events via WebSockets instead of short polling.
- **Mobile Native Wrapper:** Create a dedicated mobile app using React Native/Expo syncing to this Next.js backend.
- **AI Recommendation Agent:** Build an AI recommendation agent that analyzes a user's feed and redemptions to suggest new artists using the Litellm setup.
- **Dynamic Ticketing:** Integrate a primary ticketing provider API to sell tickets directly natively instead of linking out to Resident Advisor.
- **Blockchain Merch:** Mint limited vinyl run metadata to a blockchain for verifiable ownership.
